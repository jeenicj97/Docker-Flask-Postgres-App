# Flask PostgreSQL Docker Project

## Description
Simple Flask application running with PostgreSQL using Docker Compose.

## Services
- Flask Application
- PostgreSQL Database

## Start

docker compose up -d --build

## Stop

docker compose down

## Environment Variables

POSTGRES_DB
POSTGRES_USER
POSTGRES_PASSWORD
DB_HOST
DB_NAME
DB_USER
DB_PASSWORD

## Application URL

http://localhost:5000
